Financial Command Center setup

Follow the guided steps to connect Stripe, Plaid, and Xero credentials securely. The wizard stores secrets using the encrypted configuration manager.

1. Stripe credentials

Add a restricted Stripe API key with read/write access. The wizard saves the key encrypted on disk.

  • Paste your secret API key (sk_live_...)
  • Optionally add the publishable key for client demos
  • Use the “Test connection” button to validate permissions

2. Xero OAuth credentials

Provide your Xero client ID and secret. The wizard guides you through authenticating the organisation.

  • Enter the client ID and secret generated in the Xero developer portal
  • Use the callback button to sign in and approve access
  • The wizard stores the refresh token and tenant details automatically

3. Plaid optional configuration

Skip this step for demo mode or add sandbox keys to show Plaid workflows.

Set PLAID_CLIENT_ID and PLAID_SECRET if you plan to demo banking data.

4. Review & save

The wizard summarises configuration choices and lets you download a backup of encrypted secrets.

Use the "Save configuration" button to persist credentials and restart integrations.

Helpful endpoints

  • /health — verify the health dashboard after saving credentials.
  • /admin/create-demo-key — generate API keys for Claude or Warp.
  • /admin/ssl-help — install the local CA certificate for trusted demos.
  • /claude/setup — connect AI copilots once connectors are configured.
